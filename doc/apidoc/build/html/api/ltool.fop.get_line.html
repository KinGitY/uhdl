get_line¶

ltool.fop.get_line(file_name, match='.*', comment_prefix='//')[源代码]¶

这个函数用来在一个文件/多个文件中过滤出符合匹配条件的文本行

参数

  • file_name (str/list[str]) – file_name是将要被搜索的文件的路径名，或由被搜索的多个路径名构成的list，可以是相对路径、绝对路径，或这两者的混用

  • match (str/list[str]) – match是搜索文件时用来进行过滤的正则表达式，或由多个正则表达式构成的list，当搜索到的文件符合正则表达式 之一时将会被返回，否则会被忽略

  • comment_prefix (str) – comment_prefix是注释的前缀，加在函数自动生成的注释本文之前

返回

本函数的返回类型为list[str],这个list的元素是本文中的一行

引发

keyError – raises an exception
