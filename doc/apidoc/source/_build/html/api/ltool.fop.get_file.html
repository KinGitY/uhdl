get_file¶

ltool.fop.get_file(dir_path='./', abs_path=False, match='.*', recursion=False)[源代码]¶

这个函数用来获取一个/多个文件夹下的符合名称匹配的文件路径

参数

  • dir_path (str/list[str]) – dir_path是将要被搜索的目录的路径，或由被搜索的多个路径构成的list，可以是相对路径、绝对路径，或两者的混用

  • abs_path (bool) – abs_path是是否返回绝对路径的开关，当为True时返回绝对路径，为False时返回相对路径

  • match (str/list[str]) – match是搜索文件时用来进行过滤的正则表达式，或由多个正则表达式构成的list，当搜索到的文件符合正则表达式 之一时将会被返回，否则会被忽略

  • recursion (bool) – recurion是是否进行递归搜索的开关，当为True时，会在dir_path下进行递归搜索，即也会搜索dir_path内的子文件夹及它们 的子文件夹，当为False时，只会搜索dir下的文件

  • file_name (str/list[str]) – file_name是将要被搜索的文件的路径，或将要被搜索的多个文件的路径构成的list，可以是相对路径、绝对路径，或两者的混用

返回

本函数的返回类型为list[str],这个list中是所有的符合条件的文件路径。
