explain_file¶

ltool.pdp.explain_file(file_path, cfg)[源代码]¶

解释file_path路径指明的文件，使用cfg进行展开，并输出文本list

参数

  • file_path (str) – file_path是将被解释的源文本的文件路径，内部包含的pdp格式的文本将会被使用cfg进行解释

  • cfg – 用于对源文本进行解释的对象，必须包含有源文本中使用的属性

返回

本函数的返回类型为list[str],其内容为将源文本使用cfg进行解释后的结果
